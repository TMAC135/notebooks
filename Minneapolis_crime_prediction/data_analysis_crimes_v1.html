from math import radians, cos, sin, asin, sqrt
def cal_distance_for_great_circle(x1,x2):
    """
    Calculate the great circle distance between two points 
    on the earth (specified in decimal degrees)
    """
    lon1 = x1[0]
    lat1 = x1[1]
    lon2 = x2[0]
    lat2 = x2[1]
    # convert decimal degrees to radians 
    lon1, lat1, lon2, lat2 = map(radians, [lon1, lat1, lon2, lat2])
    # haversine formula 
    dlon = lon2 - lon1 
    dlat = lat2 - lat1 
    a = sin(dlat/2)**2 + cos(lat1) * cos(lat2) * sin(dlon/2)**2
    c = 2 * asin(sqrt(a)) 
    res = 6367 * c
    
    # here if the distance is greater 100, we may think it is unresonable
#     if res > 200:
#         return 0
    return res

Small conclusion¶

- The theft is the most common crime in Minneapolis
- Downtown West is the most 'dangerous' area, literally speaking
- Summer times seems like more crimes
- Simple KNN model could predict the model type based on the previous crime records

Still, there are more needed to investigate:

- How to add the time feature to the model?
- How can we predict the crime probability using classification model like logistic regression, softmax regression?
- Any other useful features we can use?
